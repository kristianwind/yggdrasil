Yggdrasil Panel

A self-hosted game & app server control panel for Debian/Ubuntu — think AMP + Pterodactyl, but radically easier to install, update and maintain. And not just for games: run databases and homelab apps from the same one-click Runes.

Get started → View on GitHub
curl -fsSL https://raw.githubusercontent.com/kristianwind/yggdrasil/main/install.sh | sudo bash

Everything in one binary

A single static Go binary with the web UI embedded — no separate database, no Redis, no required reverse proxy. The installer handles Docker and everything else.

📦

One binary, one command

One static binary + embedded PWA + SQLite. Install with a single command; updating is just swap & restart.

ᚱ

Runes for everything

A game or app = one declarative YAML “Rune”. Minecraft, Rust, DayZ built in; a library of community runes (databases + homelab apps) one click away.

🖥️

Real-time console & stats

Live WebSocket console with command input, log streaming, per-server CPU/RAM limits and monitoring, and a clear installing → running → stopped lifecycle.

🌐

Networking that just works

Automatic port-forwarding (UPnP / UniFi), per-server subdomains via Nginx Proxy Manager or Cloudflare Tunnel, an “online from outside” check, and connect addresses per server.

💾

Backups & scheduler

Backups to local / SFTP / SMB with retention & restore. Cron schedules for restarts, updates, backups, console commands and in-game warnings — with a run log.

🎮

Live admin & moderation

Safe restarts with in-game countdowns, an auto-restart toggle, a watchdog that auto-heals a hung server, world wipes, and a live Players tab with kick / broadcast / lock — plus an activity feed of joins, deaths and kills. All rune-declared.

🤖

AI assistant optional

Bring your own LLM (OpenAI, Anthropic, DeepSeek, Ollama, …). Get a plain-language “what happened while I was away” digest of the activity log with anomaly flags. Opt-in, advisory, and it never touches a server by itself.

🔐

Access & security

Multiple admins with scoped permissions, optional 2FA, API tokens, a full audit log. Secrets encrypted at rest (AES-256-GCM); argon2id passwords.

Minecraft Java & Bedrock Rust DayZ + Workshop mods Vaultwarden Gitea Uptime Kuma Grafana Nextcloud PostgreSQL / MariaDB / MongoDB …and a growing catalog

See it in action

Installable PWA, dark mode, mobile-friendly — works as an app on iOS/Android.

Get started

On a fresh Debian/Ubuntu box, run one command. The installer detects your distro, installs Docker if missing, creates a service user, and prints your login URL + admin password.

curl -fsSL https://raw.githubusercontent.com/kristianwind/yggdrasil/main/install.sh | sudo bash
